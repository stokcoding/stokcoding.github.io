Aplikasi Point Of Sales Berbasis Dekstop (NetBeans)

Sistem Informasi

  1. Harga : Rp 500.000
  2. Bahasa : Java (NetBeans 8.2)
  3. Database : MySQL
  4. Laporan : IReport 5.6.0

Fitur Aplikasi

  1. Login Multi User
  2. Data Pengguna
  3. Data Barang
  4. Data Kategori
  5. Data Satuan
  6. Data Customer
  7. Data Supplier
  8. Transaksi Penjualan
  9. Transaksi Retur Penjualan
  10. Transaksi Pembelian
  11. Transaksi Retur Pembelian
  12. Laporan Master
  13. Laporan Transaksi
  14. Ganti Password
  15. Backup Database
  16. Logout
  17. dan lainnya.

Keuntungan yang didapat

  1. Source Code Aplikasi Lengkap
  2. Aplikasi Bebas error/bug
  3. Aplikasi masih bisa direvisi sesuai kebutuhan (ada biaya tambahan)
  4. Garansi 100% Uang Kembali (jika aplikasi error)
  5. Dukungan & Konsultasi Gratis selama tidak ada revisi.

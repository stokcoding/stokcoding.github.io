Aplikasi E-Tiketing Bus Berbasis Web (Codeigniter)

Sistem Informasi

  1. Bahasa : Codeigniter
  2. Database : MySQL
  3. Developer : bahyusanciko (Github)

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file Aplikasi dan Database menggunakan aplikasi WinRAR.
  3. Rename folder file Aplikasi menjadi tiket, lalu copas ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama dbtiket.
  7. Import database aplikasi ke dalam database dbtiket.
  8. Jalankan project dengan ketik localhost/tiket/backend.
  9. Login dengan username : admin, Password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Muli User
  2. Pendaftaran User
  3. Cek Tiket Bus
  4. Lokasi dan Jadwal Bus
  5. List Order Tiket
  6. List Tiket dan konfirmasi
  7. Data Jadwal
  8. Data Lokasi
  9. Logout
  10. dan lainnya.

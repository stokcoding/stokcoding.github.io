Sistem Informasi E-Learning Berbasis Web (Codeigniter)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder elearning, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama elearning.
  7. Import database aplikasi ke dalam database elearning.
  8. Jalankan project dengan ketik localhost/elearning.
  9. Login dengan username : admin, password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Dashboard
  3. Data Mahasiswa
  4. Data Dosen
  5. Data Kelas
  6. Data Mata Kuliah
  7. Data Program Studi
  8. Data Laporan
  9. Data Statistik
  10. LogOut
  11. dan lainnya.

Aplikasi Pengajuan Kartu Pencari Kerja Berbasis Web (Codeigniter)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder pengajuan, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama db_pengajuan.
  7. Import database aplikasi ke dalam database db_pengajuan.
  8. Jalankan project dengan ketik localhost/pengajuan/login/login_perusahaan.
  9. Login dengan username : admin, password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Register User
  3. Halaman FrontEnd
  4. Lowongan Pekerjaan
  5. Dashboard
  6. Form Pengajuan
  7. Daftar Pencari Kerja
  8. Data Perusahaan
  9. Ganti Password
  10. Logout
  11. dan lainnya.

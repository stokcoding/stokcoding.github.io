Sistem Informasi Penjadwalan Kelas Berbasis Web (Codeigniter)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder penjadwalan, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama penjadwalan.
  7. Import database aplikasi ke dalam database penjadwalan.
  8. Jalankan project dengan ketik localhost/penjadwalan.
  9. Login dengan username : admin, password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Admin
  2. Dashboard
  3. Data Jurusan
  4. Data Kelas
  5. Data Guru
  6. Data Mapel
  7. Range Jam
  8. Jadwal Khusus
  9. Penjadwalan
  10. LogOut
  11. dan lainnya.

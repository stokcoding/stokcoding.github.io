Aplikasi Kependudukan Berbasis Web (Codeigniter)

Sistem Informasi

  1. Bahasa : Codeigniter
  2. Database : MySQL

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file Aplikasi dan Database menggunakan aplikasi WinRAR.
  3. Rename folder file Aplikasi menjadi apdp, lalu copas ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama db_penduduk.
  7. Import database aplikasi ke dalam database db_penduduk.
  8. Jalankan project dengan ketik localhost/apdp.
  9. Login dengan username : admin, Password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Admin
  2. Data Pengguna
  3. Data KTP
  4. Data Akta
  5. Data Keluarga
  6. Data Kematian
  7. Data Pendatang
  8. Data Pindah
  9. Laporan-laporan
  10. Logout
  11. dan lainnya.

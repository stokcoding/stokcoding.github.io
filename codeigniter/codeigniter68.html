Aplikasi Penjualan Apotek Berbasis Web (Codeigniter)

Sistem Requirement

  1. Bahasa : PHP 7.4
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder apotek, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama apotek.
  7. Import database aplikasi ke dalam database apotek.
  8. Jalankan project dengan ketik localhost/apotek .
  9. Login dengan username : admin, password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Admin
  2. Dashboard
  3. Data Penjualan
  4. Transaksi Penjualan
  5. Data Kasir
  6. Data Order
  7. Data Produk
  8. Data Pelanggan
  9. Data Admin
  10. Logout
  11. dan lainnya.

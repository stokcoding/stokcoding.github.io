Aplikasi Inventaris Laboratorium Berbasis Web (Codeigniter)

Sistem Requirement

  1. Bahasa : PHP 7.2
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder mekp, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama mekp.
  7. Import database aplikasi ke dalam database mekp.
  8. Jalankan project dengan ketik localhost/mekp.
  9. Login dengan email : admin@admin.com, password : 123456.
  10. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Dashboard
  3. Data Master
  4. Data Perawatan
  5. Data Laporan
  6. Data Profil
  7. Ubah Password
  8. Settings Menu
  9. LogOut
  10. dan lainnya.

Aplikasi Kasir Restoran Berbasis Web (Codeigniter)

Sistem Informasi

  1. Bahasa : Codeigniter
  2. Database : MySQL
  3. Developer : yusda08 (Github)

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp/Wampserver di Komputer.
  2. Ekstrak file Aplikasi dan Database menggunakan aplikasi WinRAR.
  3. Rename folder file Aplikasi menjadi kasir_coffe, lalu copas ke folder htdocs/www.
  4. Aktifkan Apache dan MySQL pada Xampp/Wampserver.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama kasir_coffe.
  7. Import database aplikasi ke dalam database kasir_coffe.
  8. Jalankan project dengan ketik localhost/kasir_coffe/.
  9. Login admin dengan username : admin, Password : admin.
  10. Selesai.

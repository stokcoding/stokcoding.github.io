Aplikasi Pemesanan Tiket Pesawat (Codeigniter)

Sistem Informasi

  1. Bahasa : Codeigniter
  2. Database : MySQL
  3. Developer : riskidev (Github)

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp/Wampserver di Komputer.
  2. Ekstrak file Aplikasi dan Database menggunakan aplikasi WinRAR.
  3. Rename folder file Aplikasi menjadi tiketing-pesawat, lalu copas ke folder htdocs/www.
  4. Aktifkan Apache dan MySQL pada Xampp/Wampserver.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama tiketing-pesawat_db.
  7. Import database aplikasi ke dalam database tiketing-pesawat_db.
  8. Jalankan project dengan ketik localhost/tiketing-pesawat/sign_in/p/view.
  9. Login dengan username : admin, Password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Halaman Frontend
  3. Cari Tiket Pesawat
  4. Jadwal Penerbangan

  12. dan lainnya.

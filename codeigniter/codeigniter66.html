Sistem Informasi Perjalanan Dinas Berbasis Web (Codeigniter)

Sistem Requirement

  1. Bahasa : PHP 7.2
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder sppd, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama sppd.
  7. Import database aplikasi ke dalam database sppd.
  8. Jalankan project dengan ketik localhost/sppd .
  9. Login dengan nip : admin, password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Dashboard
  3. Data User
  4. Data Pegawai
  5. Data Program
  6. Data Kegiatan
  7. Data Pekerjaan
  8. Permohonan SPPD
  9. Data Laporan
  10. Logout
  11. dan lainnya.

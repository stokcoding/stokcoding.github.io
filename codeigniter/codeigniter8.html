Aplikasi Managemen Kepanitian Berbasis Web (Codeigniter)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder panitia, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama panitia.
  7. Import database aplikasi ke dalam database panitia.
  8. Jalankan project dengan ketik localhost/panitia/index.php/admin.
  9. Login dengan username : admin, password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Admin
  2. Dashboard
  3. Data Staff
  4. Data PenanggungJawab
  5. Data Kepanitian
  6. LogOut
  7. dan lainnya.

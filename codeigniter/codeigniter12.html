Sistem Aplikasi Rental Mobil Berbasis Web (Codeigniter)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder rental_mobil, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama rental_mobil.
  7. Import database aplikasi ke dalam database rental_mobil.
  8. Jalankan project dengan ketik localhost/rental_mobil.
  9. Login dengan username : admin, password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Halaman Frontend
  2. Login Multi user
  3. Dashboard
  4. Data Mobil
  5. Data Customer
  6. Data Tipe
  7. Data Mobil
  8. Data Transaksi
  9. Data Laporan
  10. Ganti Password
  11. LogOut
  12. dan lainnya.

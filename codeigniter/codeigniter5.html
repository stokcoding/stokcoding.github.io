Aplikasi Persediaan Barang Berbasis Web (Codeigniter)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder persediaan, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama db_persediaan.
  7. Import database aplikasi ke dalam database db_persediaan.
  8. Jalankan project dengan ketik localhost/persediaan.
  9. Login dengan username : petugas, password : petugas.
  10. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Dashboard
  3. Data Barang
  4. Transaksi Barang Masuk
  5. Cetak Laporan Barang
  6. Data Ruangan
  7. Data Kondisi
  8. Data User
  9. Logout
  10. dan lainnya.

Aplikasi Penjadwalan Terapi Anak Berbasis Web (Codeigniter)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder terapi, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama terapi.
  7. Import database aplikasi ke dalam database terapi.
  8. Jalankan project dengan ketik localhost/terapi .
  9. Login dengan username : admin, password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Dashboard
  3. Data Guru
  4. Data Izin
  5. Anak Terapi
  6. Jadwal Terapi
  7. Data Terapis
  8. LogOut
  9. dan lainnya.

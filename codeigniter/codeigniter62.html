Aplikasi Managemen Data Pemilih Tetap KPU (Codeigniter)

Sistem Informasi

  1. Bahasa : Codeigniter
  2. Database : MySQL
  3. Developer : gedelumbung (Github)

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp/Wampserver di Komputer.
  2. Ekstrak file Aplikasi dan Database menggunakan aplikasi WinRAR.
  3. Rename folder file Aplikasi menjadi kpu, lalu copas ke folder htdocs/www.
  4. Aktifkan Apache dan MySQL pada Xampp/Wampserver.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama db_kpu.
  7. Import database aplikasi ke dalam database db_kpu.
  8. Jalankan project dengan ketik localhost/kpu.
  9. Login dengan username : gedelumbung, Password : gedelumbung.
  10. Selesai.

Fitur Aplikasi

  1. Halaman Frontend
  2. Login Admin
  3. Daftar Pemilih Tetap
  4. Data Kabupaten
  5. Data Kecamatan
  6. Data Kelurahan/Desa
  7. Data TPS

  9. Managemen User

  11. dan lainnya.

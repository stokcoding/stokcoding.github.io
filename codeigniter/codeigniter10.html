Aplikasi Bimbingan Konseling Berbasis Web (Codeigniter)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder konseling, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama db_konseling.
  7. Import database aplikasi ke dalam database db_konseling.
  8. Jalankan project dengan ketik localhost/konseling.
  9. Login dengan username : G000000001, password : G000000001.
  10. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Register User
  3. Dashboard
  4. Data Admin
  5. Data Guru

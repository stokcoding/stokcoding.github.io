Aplikasi Absensi Siswa SMK Berbasis Web (Codeigniter)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder absensi, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama absensi.
  7. Import database aplikasi ke dalam database absensi.
  8. Jalankan project dengan ketik localhost/absensi.
  9. Login dengan username : admin, password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Multi user
  2. Dashboard
  3. Data Kelas
  4. Data Pelajaran
  5. Data Guru
  6. Data Siswa
  7. Data Sekretaris
  8. Data Absensi
  9. Rekap Absensi
  10. LogOut
  11. dan lainnya.

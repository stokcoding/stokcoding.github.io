Aplikasi Administrasi Guru Berbasis Web (Codeigniter)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder administrasi_guru, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama db_admin.
  7. Import database aplikasi ke dalam database db_admin.
  8. Jalankan project dengan ketik localhost/administrasi_guru.
  9. Login dengan username : admin, password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Dashboard
  3. Data Guru
  4. Data Siswa
  5. Data Kelas
  6. Data Pelajaran
  7. Data Mengajar
  8. Data Jurusan
  9. Data User
  10. Ganti Password
  11. Managemen Menu
  12. LogOut
  13. dan lainnya.

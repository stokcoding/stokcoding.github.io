Aplikasi Rekrutmen Karyawan Berbasis Web (Codeigniter)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder raharja, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama raharja.
  7. Import database aplikasi ke dalam database raharja.
  8. Jalankan project dengan ketik localhost/raharja.
  9. Login dengan username : admin, password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Register User
  3. Data Peserta
  4. Data Karir
  5. Data Lowongan
  6. Tes Karyawan
  7. Hasil Tes Online
  8. LogOut
  9. dan lainnya.

Aplikasi Managemen Sekolah Berbasis Web (Codeigniter)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder ci_sekolah, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama ci_sekolah.
  7. Import database aplikasi ke dalam database ci_sekolah.
  8. Jalankan project dengan ketik localhost/ci_sekolah.
  9. Login dengan email : admin@gmail.com, password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Admin
  2. Halaman Depan
  3. Dashboard
  4. Data Pengguna
  5. Data Guru
  6. Data Murid
  7. Data Pelajaran
  8. Data Kelas
  9. Jadwal Pelajaran
  10. Data Nilai Siswa
  11. Managemen Website
  12. Hak Akses
  13. LogOut
  14. dan lainnya.

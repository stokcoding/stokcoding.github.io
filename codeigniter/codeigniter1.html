Aplikasi Penjualan Dimsum Berbasis Web (Codeigniter)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder dimsum, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama dimsum.
  7. Import database aplikasi ke dalam database dimsum.
  8. Jalankan project dengan ketik localhost/dimsum.
  9. Login dengan username : admin, password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Dashboard
  3. Data Transaksi
  4. Data Pembayaran
  5. Managemen Data
  6. Data Pengeluaran
  7. Data Laporan
  8. Data Log
  9. Logout
  10. dan lainnya.

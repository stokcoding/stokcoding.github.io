Sistem Aplikasi Managemen Kepegawaian (Codeigniter)

Sistem Informasi

  1. Bahasa : Codeigniter
  2. Database : MySQL
  3. Developer : heruprambadi (Github)

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp/Wampserver di Komputer.
  2. Ekstrak file Aplikasi dan Database menggunakan aplikasi WinRAR.
  3. Rename folder file Aplikasi menjadi ci2_simpeg, lalu copas ke folder htdocs/www.
  4. Aktifkan Apache dan MySQL pada Xampp/Wampserver.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama simpeg.
  7. Import database aplikasi ke dalam database simpeg.
  8. Jalankan project dengan ketik localhost/ci2_simpeg.
  9. Login dengan username : admin, Password : admin.
  10. Selesai.

Sistem Aplikasi Restoran Berbasis Web (Codeigniter)

Sistem Informasi

  1. Bahasa : Codeigniter
  2. Database : MySQL
  3. Developer : kanjengsaifu (Github)

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp/Wampserver di Komputer.
  2. Ekstrak file Aplikasi dan Database menggunakan aplikasi WinRAR.
  3. Rename folder file Aplikasi menjadi myresto, lalu copas ke folder htdocs/www.
  4. Aktifkan Apache dan MySQL pada Xampp/Wampserver.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama steakdb.
  7. Import database aplikasi ke dalam database steakdb.
  8. Jalankan project dengan ketik localhost/myresto.
  9. Login dengan email : admin, Password : admin.
  10. Selesai.

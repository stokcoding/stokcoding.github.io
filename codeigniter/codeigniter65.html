Aplikasi Jasa Wedding Organizer Berbasis Web (PHP)

Sistem Requirement

  1. Bahasa : PHP 7.2
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder wedding, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama wedding.
  7. Import database aplikasi ke dalam database wedding.
  8. Jalankan project dengan ketik localhost/wedding .
  9. Login dengan email : admin@gmail.com, password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Register User
  3. Dashboard
  4. Data User
  5. Data Master
  6. Layanan atau Jasa
  7. Pesanan Masuk
  8. Histori Pesanan
  9. Tentang Kami
  10. Logout
  11. dan lainnya.

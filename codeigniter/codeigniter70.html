Sistem Informasi Koperasi Simpan Pinjam (Codeigniter)

Sistem Requirement

  1. Bahasa : PHP 7.4
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder koperasi, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama koperasi.
  7. Import database aplikasi ke dalam database koperasi.
  8. Jalankan project dengan ketik localhost/koperasi .
  9. Login dengan username : admin, password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Dashboard
  3. Data Anggota
  4. Data Pengguna
  5. Data Kategori
  6. Data Simpan
  7. Data Pinjam
  8. Data Laporan
  9. Ubah Password
  10. Logout
  11. dan lainnya.

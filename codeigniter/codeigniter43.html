Sistem Informasi Akademik Berbasis Web (Codeigniter)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder mysiakad, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama db_siakad.
  7. Import database aplikasi ke dalam database db_siakad.
  8. Jalankan project dengan ketik localhost/mysiakad.
  9. Login dengan username : admin, password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Dashboard
  3. Data Master
  4. Data Akademik
  5. Data Civitas
  6. Data Transaksi
  7. Data Laporan
  8. Data Grafik
  9. LogOut
  10. dan lainnya.

Sistem Informasi Inventory Berbasis Web (Codeigniter)

Sistem Informasi

  1. Bahasa : Codeigniter
  2. Database : MySQL
  3. Developer : kaconk17 (Github)

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp/Wampserver di Komputer.
  2. Ekstrak file Aplikasi dan Database menggunakan aplikasi WinRAR.
  3. Rename folder file Aplikasi menjadi inventory, lalu copas ke folder htdocs/www.
  4. Aktifkan Apache dan MySQL pada Xampp/Wampserver.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama inventory.
  7. Import database aplikasi ke dalam database inventory.
  8. Jalankan project dengan ketik localhost/inventory.
  9. Login dengan email : admin, Password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Barang Masuk
  3. Barang Keluar
  4. Stok Barang
  5. Permintaan Barang
  6. Pembelian Barang
  7. Laporan-laporan
  8. Logout
  9. dan lainnya.

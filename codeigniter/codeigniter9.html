Aplikasi Arsip Data Faktur Berbasis Web (Codeigniter)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder faktur, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama faktur.
  7. Import database aplikasi ke dalam database faktur.
  8. Jalankan project dengan ketik localhost/faktur.
  9. Login dengan username : tazkia, password : tazkia.
  10. Selesai.

Fitur Aplikasi

  1. Login Admin
  2. Dashboard
  3. Data Faktur
  4. Data Produk
  5. Data User
  6. Data History
  7. LogOut
  8. dan lainnya.

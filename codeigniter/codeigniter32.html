Aplikasi Penjadwalan Perekrutan Karyawan (Codeigniter)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder jadwal, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama jadwal.
  7. Import database aplikasi ke dalam database jadwal.
  8. Jalankan project dengan ketik localhost/jadwal.
  9. Login dengan username : brohid, password : brohid.
  10. Selesai.

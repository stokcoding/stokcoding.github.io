Sistem Informasi Rental Mobil Berbasis Web (Codeigniter)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder rental-mobil-ci, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama rental-mobil-ci.
  7. Import database aplikasi ke dalam database rental-mobil-ci.
  8. Jalankan project dengan ketik localhost/rental-mobil-ci .
  9. Login dengan username : ridho123, password : 123.
  10. Selesai.

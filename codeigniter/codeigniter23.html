Aplikasi Toko Baju Berbasis Web (Codeigniter)

Sistem Requirement

  1. Bahasa : PHP 8.0
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder btis-apps, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama btis.
  7. Import database aplikasi ke dalam database btis.
  8. Jalankan project dengan ketik localhost/btis-apps.
  9. Login dengan email : admin@gmail.com, password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Halaman Depan
  3. Dashboard
  4. Katalog Produk
  5. Kategori Produk
  6. Keranjang Belanja
  7. Produk Favorit
  8. Data Pesanan
  9. Produk Diskon
  10. LogOut
  11. dan lainnya.

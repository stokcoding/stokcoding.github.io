Aplikasi Penjualan Kasir Berbasis Web (Laravel)

Sistem Requirement

  1. Bahasa : PHP 8.2
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp dan Composer di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder pos, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama pos.
  7. Import database aplikasi ke dalam database pos.
  8. Buka terminal di dalam project, ketikan php artisan serve.
  9. Jalankan project dengan ketik http://127.0.0.1:8000.
  10. Login dengan email : admin@gmail.com, Password : 123.
  11. Selesai.

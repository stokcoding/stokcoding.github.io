Aplikasi Presensi Guru Berbasis Web (Laravel)

Sistem Informasi

  1. Bahasa : Laravel
  2. Database : MySQL
  3. Developer : FaruqRahmadani (Github)

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp/Wampserver di Komputer.
  2. Ekstrak file Aplikasi dan Database menggunakan aplikasi WinRAR.
  3. Rename folder file Aplikasi menjadi PresensiGuru, lalu copas ke folder htdocs/www.
  4. Aktifkan Apache dan MySQL pada Xampp/Wampserver.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama PresensiGuru.
  7. Import database aplikasi ke dalam database PresensiGuru.
  8. Jalankan project dengan ketik localhost/PresensiGuru/public/.
  9. Login dengan username : InitialUser, Password : InitialPassword.
  10. Selesai.

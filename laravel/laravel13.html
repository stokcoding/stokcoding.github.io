Aplikasi e-Catering Berbasis Web (Laravel)

Sistem Informasi

  1. Bahasa : Laravel
  2. Database : MySQL
  3. Developer : caesarali (Github)

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp/Wampserver di Komputer.
  2. Ekstrak file Aplikasi dan Database menggunakan aplikasi WinRAR.
  3. Rename folder file Aplikasi menjadi e-catering, lalu copas ke folder htdocs/www.
  4. Aktifkan Apache dan MySQL pada Xampp/Wampserver.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama e-catering.
  7. Import database aplikasi ke dalam database e-catering.
  8. Jalankan project dengan ketik localhost/e-catering/public/.
  9. Login dengan email : admin@catering, Password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Registrasi User
  3. Data Pemesanan
  4. Katalog Menu
  5. Data Order
  6. Data Customer
  7. Data Makanan
  8. Status Pesanan
  9. Logout
  10. dan lainnya.

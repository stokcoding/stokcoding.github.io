Aplikasi Pembelian dan Penjualan Berbasis Web (Laravel)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp dan Composer di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder MyInventory, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama my_inventory.
  7. Import database aplikasi ke dalam database my_inventory.
  8. Buka terminal di dalam project, ketikan php artisan serve.
  9. Jalankan project dengan ketik http://127.0.0.1:8000/login.
  10. Login dengan email : admin@admin.com, Password : admin.
  11. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Dashboard
  3. Data Produk
  4. Data Kategori
  5. Data Supplier
  6. Data Konsumen
  7. Transaksi Pembelian
  8. Transaksi Penjualan
  9. Data Laporan
  10. Data User
  11. LogOut
  12. dan lainnya.

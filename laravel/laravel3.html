Aplikasi Inventory Berbasis Web (Laravel)

Sistem Requirement

  1. Bahasa : PHP 7.4
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp dan Composer di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder si_inventory, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama db_inventori.
  7. Import database aplikasi ke dalam database db_inventori.
  8. Buka terminal di dalam project, ketikan php artisan serve.
  9. Jalankan project dengan ketik http://127.0.0.1:8000/login.
  10. Login dengan email : admin, Password : admin.
  11. Selesai.

Fitur Aplikasi

  1. Login Admin
  2. Dashboard
  3. Data Karyawan
  4. Data Barang
  5. Data Kategori
  6. Data Unit
  7. Pengambilan Barang
  8. Laporan Pengambilan
  9. LogOut
  10. dan lainnya.

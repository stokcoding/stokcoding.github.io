Sistem Informasi Gudang Berbasis Web (Laravel)

Sistem Requirement

  1. Bahasa : PHP 7.4
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp dan Composer di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder si_gudang, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama db_gudang.
  7. Import database aplikasi ke dalam database db_gudang.
  8. Buka terminal di dalam project, ketikan php artisan serve.
  9. Jalankan project dengan ketik http://127.0.0.1:8000/login.
  10. Login dengan email : admin, Password : admin.
  11. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Dashboard
  3. Data Karyawan
  4. Data Pemasok
  5. Data Barang
  6. Barang Masuk
  7. Barang Keluar
  8. Data Kategori
  9. Data Satuan
  10. Data User
  11. Laporan-laporan
  12. LogOut
  13. dan lainnya.

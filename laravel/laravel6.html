Aplikasi Perpustakaan Berbasis Web (Laravel)

Sistem Informasi

  1. Bahasa : Laravel
  2. Database : MySQL
  3. Developer : mahmudinm (Github)

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp/Wampserver di Komputer.
  2. Ekstrak file Aplikasi dan Database menggunakan aplikasi WinRAR.
  3. Rename folder file Aplikasi menjadi laravel-perpustakaan, lalu copas ke folder htdocs/www.
  4. Aktifkan Apache dan MySQL pada Xampp/Wampserver.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama laravel_perpustakaan.
  7. Import database aplikasi ke dalam database laravel_perpustakaan.
  8. Jalankan project dengan ketik localhost/laravel-perpustakaan/public.
  9. Login dengan username : admin, Password : password.
  10. Selesai.

Fitur Aplikasi

  1. Login Admin
  2. Data Buku
  3. Data User
  4. Data Peminjaman
  5. Logout
  6. dan lainnya.

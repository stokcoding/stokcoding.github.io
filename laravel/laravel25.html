Aplikasi Pemilihan Ketua Osis Berbasis Web (Laravel)

Sistem Informasi

  1. Bahasa : Framework Laravel
  2. Database : MySQL
  3. Developer : fikrisuheri (Github)

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp dan Composer di Komputer.
  2. Ekstrak file Aplikasi dan Database menggunakan aplikasi WinRAR.
  3. Rename folder file Aplikasi menjadi laravel-evoting, lalu copas ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama laravel_pilketos.
  7. Import database aplikasi ke dalam database laravel_pilketos.
  8. Buka terminal di dalam project, ketikan php artisan serve.
  9. Jalankan project dengan ketik http://127.0.0.1:8000/.
  10. Login dengan email : admin@gmail.com, Password : rahasia.
  11. Selesai.

  3. Developer : indrawind48 (Github)

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp/Wampserver di Komputer.
  2. Ekstrak file Aplikasi dan Database menggunakan aplikasi WinRAR.
  3. Rename folder file Aplikasi menjadi lab, lalu copas ke folder htdocs/www.
  4. Aktifkan Apache dan MySQL pada Xampp/Wampserver.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama klin.
  7. Import database aplikasi ke dalam database klin.
  8. Jalankan project dengan ketik localhost/lab.
  9. Login dengan admin : admin, Password : admin.
  10. Selesai.

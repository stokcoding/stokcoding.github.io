Aplikasi E-Learning Berbasis Web (PHP)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder smkn2wnsb, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama smkn2wnsb.
  7. Import database aplikasi ke dalam database smkn2wnsb.
  8. Jalankan project dengan ketik localhost/smkn2wnsb.
  9. Login dengan username : admin, password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Dashboard
  3. Data Guru
  4. Data Siswa
  5. Data Kelas
  6. Data Jurusan
  7. Data Materi
  8. Data Tugas
  9. Data Ujian
  10. Data Nilai
  11. LogOut
  12. dan lainnya.

Sistem Informasi Aset Kantor Berbasis Web (PHP)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder aset_kantor, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama aset_kantor.
  7. Import database aplikasi ke dalam database aset_kantor.
  8. Jalankan project dengan ketik localhost/aset_kantor.
  9. Login dengan username : admin, password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Dashboard
  3. Data Grafik
  4. Data Barang
  5. Data Furniture
  6. Data Kendaraan
  7. Data Elektronik
  8. Cetak Laporan
  9. Data Panduan
  10. Bantuan
  11. LogOut
  12. dan lainnya.

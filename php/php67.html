Aplikasi Pelayanan Servis Komputer Berbasis Web (PHP)

Sistem Informasi

  1. Bahasa : PHP
  2. Database : MySQL
  3. Developer : greatsman (Github)

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp/Wampserver di Komputer.
  2. Ekstrak file Aplikasi dan Database menggunakan aplikasi WinRAR.
  3. Rename folder file Aplikasi menjadi pelayanan, lalu copas ke folder htdocs/www.
  4. Aktifkan Apache dan MySQL pada Xampp/Wampserver.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama pelayanan.
  7. Import database aplikasi ke dalam database pelayanan.
  8. Jalankan project dengan ketik localhost/pelayanan/login.
  9. Login dengan username : agung123, Password : 123.
  10. Selesai.

Aplikasi Managemen Data Rumah Sakit (PHP)

Sistem Informasi

  1. Bahasa : PHP
  2. Database : MySQL
  3. Developer : maful (Github)

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file Aplikasi dan Database menggunakan aplikasi WinRAR.
  3. Rename folder file Aplikasi menjadi gd_sirs, lalu copas ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama gd_sirs.
  7. Import database aplikasi ke dalam database gd_sirs.
  8. Jalankan project dengan ketik localhost/gd_sirs.
  9. Login dengan username : maful, Password : maful.
  10. Selesai.

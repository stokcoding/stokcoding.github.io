Aplikasi CMS Portal Berita Berbasis Web (PHP)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder portal-berita, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama portal_berita.
  7. Import database aplikasi ke dalam database portal_berita.
  8. Jalankan project dengan ketik localhost/portal-berita/admin.
  9. Login dengan username : admin, password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Halaman FrontEnd
  3. Dashboard
  4. Data Berita
  5. Kategori Berita
  6. Data Agenda
  7. Data Profil
  8. Data Download
  9. Galeri Foto
  10. Komentar Berita
  11. Data Pengumuman
  12. LogOut
  13. dan lainnya.

Sistem Informasi Rental Mobil Berbasis Web (PHP)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder rental-mobil-php-native, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama rental_mobil.
  7. Import database aplikasi ke dalam database rental_mobil.
  8. Jalankan project dengan ketik localhost/rental-mobil-php-native.
  9. Login dengan username : ridho, password : ridho.
  10. Selesai.

Fitur Aplikasi

  1. Login Admin
  2. Dashboard
  3. Data Order
  4. Data Mobil
  5. Data User
  6. LogOut
  7. dan lainnya.

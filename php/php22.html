Aplikasi Pendaftaran Siswa SMP Baru Berbasis Web (PHP)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder registrasismp, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama registrasismp.
  7. Import database aplikasi ke dalam database registrasismp.
  8. Jalankan project dengan ketik localhost/registrasismp/akun/akun.php.
  9. Login dengan username : admin, password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Halaman FrontEnd
  3. Dashboard
  4. Pendaftaran Siswa
  5. Data User
  6. Data Pengumuman
  7. Data Kelulusan
  8. Data Grafik
  9. Kirim Pesan
  10. LogOut
  11. dan lainnya.

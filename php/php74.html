Sistem Informasi Pengaduan Masyarakat Berbasis Web (PHP)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder pengaduan, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama pengaduan.
  7. Import database aplikasi ke dalam database pengaduan.
  8. Jalankan project dengan ketik localhost/pengaduan.
  9. Login admin dengan username : adin, password : adin.
  10. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Dashboard
  3. Tulis Laporan
  4. Tanggapi Laporan
  5. Data Petugas
  6. Data Masyarakat
  7. Data Pengaduan
  8. Data Laporan
  9. LogOut
  10. dan lainnya.

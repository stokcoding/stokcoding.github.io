Aplikasi Pendaftaran Siswa Berbasis Web (PHP)

Sistem Informasi

  1. Bahasa : PHP
  2. Database : MySQL
  3. Developer : azharirizal (Github)

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file Aplikasi dan Database menggunakan aplikasi WinRAR.
  3. Rename folder file Aplikasi menjadi psb_online, lalu copas ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama psb_online.
  7. Import database aplikasi ke dalam database psb_online.
  8. Jalankan project dengan ketik localhost/psb_online/admin/pages/login.php.
  9. Login dengan username : admin, Password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Pendaftaran Siswa
  3. Pengumuman
  4. Data Pendaftar
  5. Informasi Kelulusan
  6. Data Calon Siswa
  7. Cetak Pembayaran
  8. Cetak Data siswa
  9. dan lain sebagainya.

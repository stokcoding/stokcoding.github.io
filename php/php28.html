Sistem Informasi Laporan Keuangan Berbasis Web (PHP)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder si_pelaporan_keuangan, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama db_kampung_flory.
  7. Import database aplikasi ke dalam database db_kampung_flory.
  8. Jalankan project dengan ketik localhost/si_pelaporan_keuangan.
  9. Login dengan username : admin, password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Dashboard
  3. Data Akun
  4. Jurnal Umum
  5. Buku Besar
  6. Arus Kas
  7. Laba/Rugi
  8. Perubahan Modal
  9. Neraca
  10. Data User
  11. LogOut
  12. dan lainnya.

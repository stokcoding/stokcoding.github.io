Aplikasi Pembayaran PPOB Listrik Berbasis Web (PHP)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder Aplikasi-Pembayaran-Listrik, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama pembayaran_listrik.
  7. Import database aplikasi ke dalam database pembayaran_listrik.
  8. Jalankan project dengan ketik localhost/Aplikasi-Pembayaran-Listrik/ppob/login.php.
  9. Login dengan username : manager, password : 123.
  10. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Data Tarif
  3. Data Pelanggan
  4. Data Teller
  5. Data Pelanggan
  6. Tagihan Pelanggan
  7. Pengelolaan Tagihan
  8. Laporan-Laporan
  9. LogOut
  10. dan lainnya.

Aplikasi Ujian Online Berbasis Web (Codeigniter)

Sistem Informasi

  1. Bahasa : PHP
  2. Database : MySQL

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file Aplikasi dan Database menggunakan aplikasi WinRAR.
  3. Rename folder file Aplikasi menjadi ujianonline, lalu copas ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama dbujianonline.
  7. Import database aplikasi ke dalam database dbujianonline.
  8. Jalankan project dengan ketik localhost/ujianonline.
  9. Login dengan username : admin, Password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Data Guru
  3. Data Siswa
  4. Data Soal Ujian
  5. Data Peserta Ujian
  6. Data Hasil Ujian
  7. Utilites
  8. Ganti Password
  9. LogOut
  10. dan lainnya.

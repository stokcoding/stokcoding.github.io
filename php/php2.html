Aplikasi Pengelolaan Uang Kas Sekolah Berbasis Web (PHP)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder uang-kas, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama uang_kas.
  7. Import database aplikasi ke dalam database uang_kas.
  8. Jalankan project dengan ketik localhost/uang-kas.
  9. Login dengan username : andri123, password : 123456.
  10. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Dashboard
  3. Data Jabatan
  4. Data User
  5. Data Siswa
  6. Data Uang Kas
  7. Data Pengeluaran
  8. Data Laporan
  9. Data Riwayat
  10. Logout
  11. dan lainnya.

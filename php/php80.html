Aplikasi Konsultasi Dokter Berbasis Web (PHP)

Sistem Requirement

  1. Bahasa : PHP 7.4
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder konsuldoc, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama konsuldoc.
  7. Import database aplikasi ke dalam database konsuldoc.
  8. Jalankan project dengan ketik localhost/konsuldoc.
  9. Login admin dengan email : admin@gmail.com, password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Register User
  3. Halaman Frontend
  4. Dashboard
  5. Data Artikel
  6. Data Spesialis
  7. Data Dokter
  8. Data User
  9. Logout
  10. dan lainnya.

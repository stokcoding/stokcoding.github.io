Aplikasi Arsip Dokumen Cetak Berbasis Web (PHP)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder arsip, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama arsip.
  7. Import database aplikasi ke dalam database arsip.
  8. Jalankan project dengan ketik localhost/arsip.
  9. Login dengan username : adminarsip, password : adminarsip.
  10. Selesai.

Fitur Aplikasi

  1. Login Admin
  2. Dashboard
  3. Data Arsip
  4. Data Kode Arsip
  5. Data Bidang
  6. Data Akun
  7. Logout
  8. dan lainnya.

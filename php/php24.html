Sistem Informasi Inventaris Sekolah Berbasis Web (PHP)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder inventaris, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama inventaris.
  7. Import database aplikasi ke dalam database inventaris.
  8. Jalankan project dengan ketik localhost/inventaris.
  9. Login dengan username : admin, password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Register User
  3. Dashboard
  4. Data Peminjaman
  5. Data Barang
  6. Barang Masuk
  7. Barang Keluar
  8. Data Supplier
  9. Data Stok
  10. Data Lainnya
  11. LogOut
  12. dan lainnya.

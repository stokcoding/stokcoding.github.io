Sistem Informasi Mahasiswa Berbasis Web (PHP)

Sistem Requirement

  1. Bahasa : PHP 7.2
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder mahasiswa, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama mahasiswa.
  7. Import database aplikasi ke dalam database mahasiswa.
  8. Jalankan project dengan ketik localhost/mahasiswa.
  9. Login admin dengan username : admin, password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Dashboard
  3. Data Mahasiswa
  4. Data Mata Kuliah
  5. Data Jadwal Kuliah
  6. Data Pengumuman
  7. Pengaturan Profil
  8. Logout
  9. dan lainnya.

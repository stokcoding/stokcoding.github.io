Sistem Informasi

  1. Bahasa : PHP
  2. Database : MySQL
  3. Developer : bilirenaldi (Github)

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp/Wampserver di Komputer.
  2. Ekstrak file Aplikasi dan Database menggunakan aplikasi WinRAR.
  3. Rename folder file Aplikasi menjadi nprj, lalu copas ke folder htdocs/www.
  4. Aktifkan Apache dan MySQL pada Xampp/Wampserver.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama nprj.
  7. Import database aplikasi ke dalam database nprj.
  8. Jalankan project dengan ketik localhost/nprj/admin.
  9. Login dengan nim : 161410187, Password : admin.
  10. Selesai.

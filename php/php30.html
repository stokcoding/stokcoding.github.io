Aplikasi Pendaftaran Peserta Didik Baru Berbasis Web (PHP)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder ppdb, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama db_ppdb.
  7. Import database aplikasi ke dalam database db_ppdb.
  8. Jalankan project dengan ketik localhost/ppdb/etc-admin.
  9. Login dengan username : admin, password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Admin
  2. Halaman FrontEnd
  3. Dashboard
  4. Form Pendaftaran
  5. Hasil Pengumuman
  6. Data Peserta
  7. Data Siswa
  8. Pengaturan
  9. Logout
  10. dan lainnya.

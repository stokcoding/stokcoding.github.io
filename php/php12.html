Aplikasi Parkir Berbasis Web (PHP)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder parkir, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama parkir.
  7. Import database aplikasi ke dalam database parkir.
  8. Jalankan project dengan ketik localhost/parkir.
  9. Login dengan username : admin, password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Halaman Frontend
  2. Daftar Kendaraan
  3. Login Admin
  4. Dashboard
  5. Data Kendaraan
  6. Jenis Kendaraan
  7. LogOut
  8. dan lainnya.

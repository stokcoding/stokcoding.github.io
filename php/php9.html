Aplikasi SPK Santri Terbaik Metode SAW (PHP)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder saw-santri, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama db_saw.
  7. Import database aplikasi ke dalam database db_saw.
  8. Jalankan project dengan ketik localhost/saw-santri.
  9. Login dengan username : admin, password : admin123.
  10. Selesai.

Fitur Aplikasi

  1. Login Admin
  2. Input Santri
  3. Data Santri
  4. Data Normalisasi
  5. Data Hasil
  6. Data Arsip
  7. LogOut
  8. dan lainnya.

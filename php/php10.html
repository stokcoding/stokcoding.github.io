Aplikasi Toko Elektronik Berbasis Web (PHP)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder tokoonline, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama db_tokoonline.
  7. Import database aplikasi ke dalam database db_tokoonline.
  8. Jalankan project dengan ketik localhost/tokoonline.
  9. Login dengan username : admin, password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Register User
  3. Katalog Produk
  4. Keranjang Belanja
  5. Checkout Transaksi
  6. Data Produk
  7. Data Kategori
  8. Data Pembelian
  9. Data Laporan
  10. Data Pelanggan
  11. LogOut
  12. dan lainnya.

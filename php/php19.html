Aplikasi Catatan Keuangan Berbasis Web (PHP)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder keuangan, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama db_keuangan.
  7. Import database aplikasi ke dalam database db_keuangan.
  8. Jalankan project dengan ketik localhost/keuangan.
  9. Login dengan email : saiful@mail.com, password : tes123.
  10. Selesai.

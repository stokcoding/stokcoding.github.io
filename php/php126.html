Sistem Informasi Arisan Berbasis Web (PHP)

Sistem Informasi

  1. Bahasa : PHP
  2. Database : MySQL

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file Aplikasi dan Database menggunakan aplikasi WinRAR.
  3. Rename folder file Aplikasi menjadi arisan, lalu copas ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama db_arisan.
  7. Import database aplikasi ke dalam database db_arisan.
  8. Jalankan project dengan ketik localhost/arisan.
  9. Login dengan nip : 123456787654345678, Password : admin.
  10. Selesai.

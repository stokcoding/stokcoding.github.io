Aplikasi Minimarket Berbasis Web (PHP)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder minimarket, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama minimarket.
  7. Import database aplikasi ke dalam database minimarket.
  8. Jalankan project dengan ketik localhost/minimarket.
  9. Login dengan username : admin, password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Data Kasir
  3. Data Barang
  4. Data Kategori
  5. Data Supplier
  6. Data Customer
  7. Data Retur
  8. Data Laporan
  9. Data Grafik
  10. LogOut
  11. dan lainnya.

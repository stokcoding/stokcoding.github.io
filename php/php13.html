Sistem Informasi Absensi Berbasis Web (PHP)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder absensi, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama db_absensi.
  7. Import database aplikasi ke dalam database db_absensi.
  8. Jalankan project dengan ketik localhost/absensi.
  9. Login dengan username : admin, password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Admin
  2. Dashboard
  3. Data Karyawan
  4. Data Jadwal Kerja
  5. Data Absensi
  6. Laporan Absensi
  7. Logout
  8. dan lainnya.

Aplikasi Streaming Film Berbasis Web (PHP)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder streaming, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama streaming.
  7. Import database aplikasi ke dalam database streaming.
  8. Jalankan project dengan ketik localhost/streaming/admin.
  9. Login dengan username : andri123, password : 12345.
  10. Selesai.

Fitur Aplikasi

  1. Login Admin
  2. Halaman FrontEnd
  3. Dashboard
  4. Katalog Film
  5. Pencarian Film
  6. Data Genre Film
  7. Data Komentar
  8. Data Riwayat
  9. Data Profil
  10. Logout
  11. dan lainnya.

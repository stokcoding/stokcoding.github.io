Sistem Pencatatan Keluhan Pelanggan Berbasis Web (PHP)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder si_kepel, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama db_keluhan.
  7. Import database aplikasi ke dalam database db_keluhan.
  8. Jalankan project dengan ketik localhost/si_kepel.
  9. Login dengan username : admin1, password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Dashboard
  3. Data Keluhan
  4. Data Perbaikan
  5. Data Admin
  6. Data Teknisi
  7. Data Pelanggan
  8. LogOut
  9. dan lainnya.

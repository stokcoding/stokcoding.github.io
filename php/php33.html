Aplikasi Penjualan Buku Berbasis Web (PHP)

Sistem Informasi

  1. Bahasa : PHP
  2. Database : MySQL
  3. Developer : ichsanharun (Github)

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file Aplikasi dan Database menggunakan aplikasi WinRAR.
  3. Rename folder file Aplikasi menjadi penjualan, lalu copas ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama penjualan.
  7. Import database aplikasi ke dalam database penjualan.
  8. Jalankan project dengan ketik localhost/penjualan.
  9. Login dengan username : adm0001, Password : 1997.
  10. Selesai.

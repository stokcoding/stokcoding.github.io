Sistem Informasi Pariwisata Berbasis Web (PHP)

Sistem Informasi

  1. Bahasa : PHP
  2. Database : MySQL

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file Aplikasi dan Database menggunakan aplikasi WinRAR.
  3. Rename folder file Aplikasi menjadi pariwisata, lalu copas ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama dbdinaspariwisata.
  7. Import database aplikasi ke dalam database dbdinaspariwisata.
  8. Jalankan project dengan ketik localhost/pariwisata/admin.
  9. Login dengan username : admin, Password : admin.
  10. Selesai.

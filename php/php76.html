Aplikasi Pemesanan Tiket Pesawat Berbasis Web (PHP)

Sistem Informasi

  1. Bahasa : PHP
  2. Database : MySQL
  3. Developer : dewangga7111 (Github)

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp/Wampserver di Komputer.
  2. Ekstrak file Aplikasi dan Database menggunakan aplikasi WinRAR.
  3. Rename folder file Aplikasi menjadi tiket, lalu copas ke folder htdocs/www.
  4. Aktifkan Apache dan MySQL pada Xampp/Wampserver.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama tiket.
  7. Import database aplikasi ke dalam database tiket.
  8. Jalankan project dengan ketik localhost/tiket/admin.
  9. Login dengan username : admin, Password : admin.
  10. Selesai.

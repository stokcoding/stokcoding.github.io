Aplikasi Toko Kue Online Berbasis Web (PHP)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder kue, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama kue.
  7. Import database aplikasi ke dalam database kue.
  8. Jalankan project dengan ketik localhost/kue.
  9. Login admin dengan username : admin, password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Register User
  3. Halaman Frontend
  4. Dashboard
  5. Katalog Kue
  6. Keranjang Belanja
  7. Checkout
  8. Data Master
  9. Data Transaksi
  10. Data Laporan
  11. Logout
  12. dan lainnya.

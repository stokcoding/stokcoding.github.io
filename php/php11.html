Aplikasi Rekam Medis Rumah Sakit (PHP)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder rumahsakit, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama rumahsakit.
  7. Import database aplikasi ke dalam database rumahsakit.
  8. Jalankan project dengan ketik localhost/rumahsakit.
  9. Login dengan username : admin, password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Admin
  2. Data Pasien
  3. Data Dokter
  4. Data Poliklinik
  5. Data Obat
  6. Rekam Medis
  7. LogOut
  8. dan lainnya.

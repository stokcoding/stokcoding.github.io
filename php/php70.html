Sistem Informasi Pemetaan Kasus Kriminal (PHP)

Sistem Informasi

  1. Bahasa : PHP
  2. Database : MySQL
  3. Developer : andrinto (Github)

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp/Wampserver di Komputer.
  2. Ekstrak file Aplikasi dan Database menggunakan aplikasi WinRAR.
  3. Rename folder file Aplikasi menjadi kasuskriminal, lalu copas ke folder htdocs/www.
  4. Aktifkan Apache dan MySQL pada Xampp/Wampserver.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama kasuskriminal.
  7. Import database aplikasi ke dalam database kasuskriminal.
  8. Jalankan project dengan ketik localhost/kasuskriminal.
  9. Login dengan username : admin, Password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Admin
  2. Peta dengan Marker
  3. Jenis Kriminal dengan Marker
  4. Lihat Jumlah Kriminal
  5. Mengganti jenis View
  6. Data Kriminal
  7. Data Profil
  8. Data Pengguna
  9. LogOut
  10. dan lainnya.

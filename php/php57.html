Sistem Aplikasi Penyewaan Lapangan Futsal (PHP)

Sistem Informasi

  1. Bahasa : PHP
  2. Database : MySQL
  3. Developer : risqi17 (Github)

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp/Wampserver di Komputer.
  2. Ekstrak file Aplikasi dan Database menggunakan aplikasi WinRAR.
  3. Rename folder file Aplikasi menjadi gofutsal, lalu copas ke folder htdocs/www.
  4. Aktifkan Apache dan MySQL pada Xampp/Wampserver.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama gofutsal.
  7. Import database aplikasi ke dalam database gofutsal.
  8. Jalankan project dengan ketik localhost/gofutsal/admin.
  9. Login dengan username : admin, Password : admin.
  10. Selesai.

Fitur Aplikasi

Sistem Aplikasi Pembuatan CV Online (PHP)

Sistem Informasi

  1. Bahasa : PHP
  2. Database : MySQL
  3. Developer : azharirizal (Github)

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file Aplikasi dan Database menggunakan aplikasi WinRAR.
  3. Rename folder file Aplikasi menjadi cv, lalu copas ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama cv.
  7. Import database aplikasi ke dalam database cv.
  8. Jalankan project dengan ketik localhost/cv.
  9. Login dengan username : admin, Password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Admin
  2. Registrasi Akun
  3. Isi Biodata
  4. Melihat CV Biodata
  5. Pengaturan
  6. Logout
  7. dan lain sebagainya.

Aplikasi PSB Sekolah SMP Berbasis Web (PHP)

Sistem Informasi

  1. Bahasa : PHP
  2. Database : MySQL

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file Aplikasi dan Database menggunakan aplikasi WinRAR.
  3. Rename folder file Aplikasi menjadi ppdb_smp, lalu copas ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama db_smp.
  7. Import database aplikasi ke dalam database db_smp.
  8. Jalankan project dengan ketik localhost/ppdb_smp/admin.
  9. Login dengan username : admin, Password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Admin
  2. Halaman Frontend
  3. Register Siswa
  4. Data Galeri
  5. Data Pendaftar
  6. Data Member
  7. Data Calon
  8. Data Pengumuman
  9. Data Laporan
  10. dan lainnya.

Aplikasi Penggajian Guru Berbasis Web (PHP)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder penggajian, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama penggajian.
  7. Import database aplikasi ke dalam database penggajian.
  8. Jalankan project dengan ketik localhost/penggajian.
  9. Login admin dengan username : admin, password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Dashboard
  3. Data Guru
  4. Data Jabatan
  5. Data Pengguna
  6. Transaksi Penggajian
  7. Data Laporan
  8. Backup Database
  9. Logout
  10. dan lainnya.

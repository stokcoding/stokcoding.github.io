Aplikasi Penjualan Bahan Pangan Berbasis Web (PHP)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder berkat-usahaUD, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama berkat_usahaud.
  7. Import database aplikasi ke dalam database berkat_usahaud.
  8. Jalankan project dengan ketik localhost/berkat-usahaUD.
  9. Login dengan email : admin@gmail.com, password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Login Multi User
  2. Register User
  3. FrontEnd
  4. Data Produk
  5. Transaksi Penjualan
  6. Transaksi Pembelian
  7. Laporan Pembelian
  8. LogOut
  9. dan lainnya.

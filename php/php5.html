Sistem Informasi PPDB Sekolah SD Berbasis Web (PHP)

Sistem Requirement

  1. Bahasa : PHP 5.6
  2. Database : MySQL
  3. Server : Xampp

Panduan Instalasi

  1. Download dan Instal Aplikasi Xampp di Komputer.
  2. Ekstrak file menggunakan aplikasi WinRAR.
  3. Copy folder ppdb_sd, lalu paste ke folder htdocs.
  4. Aktifkan Apache dan MySQL pada Xampp.
  5. Buka browser, lalu buka alamat localhost/phpMyAdmin.
  6. Buat database baru dengan nama ppdb_sd.
  7. Import database aplikasi ke dalam database ppdb_sd.
  8. Jalankan project dengan ketik localhost/ppdb_sd.
  9. Login dengan username : admin, password : admin.
  10. Selesai.

Fitur Aplikasi

  1. Halaman Frontend
  2. Login Multi User
  3. Dashboard
  4. Data Siswa
  5. Data Orang Tua
  6. Data Administrasi
  7. Data Pengguna
  8. Cetak Kartu
  9. Pengaturan
  10. dan lainnya.
